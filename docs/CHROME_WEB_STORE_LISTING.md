## Detailed description — English

```
Netflix Cat helps you organize Netflix My List with your own custom categories — so you can find what you want to watch faster.

Netflix’s My List is great for saving titles, but it doesn’t let you group them your way. Netflix Cat adds lightweight category tagging and filtering directly on your My List page.

WHAT YOU CAN DO

• Create custom categories — e.g. “Weekend”, “With Kids”, “Finish Later”
• Tag titles on My List — assign categories directly on each card
• Filter by category — show only the titles you care about right now
• Edit and reorder categories — rename, delete, and drag to reorder from the extension popup
• Choose card size — small, medium, or large grid cards
• Export and import your data — back up categories and assignments as JSON
• Choose your language — English and Traditional Chinese (Hong Kong)
• Auto scroll scan — helpful if your list has many titles and not all are loaded yet

HOW TO USE

1. Install Netflix Cat and open Netflix My List
2. Click the Netflix Cat icon (top-right) to create categories
3. On My List, use “Manage Category” to tag titles
4. Use the category dropdown to filter your list

PRIVACY

• Works only on netflix.com
• Your categories and tags are stored locally in your browser
• No account or sign-in required for Netflix Cat itself

BETA NOTICE

Netflix Cat is in beta. Features may change as development continues. Netflix’s website layout can also change, which may affect how the extension works until an update is released.

Netflix Cat is an independent tool and is not affiliated with or endorsed by Netflix.
```

## Detailed description — 繁體中文

```
Netflix Cat 讓你用自訂分類整理 Netflix My List，更快找到想看的內容。

Netflix 的 My List 適合收藏影片，但無法按你的方式分組。Netflix Cat 在 My List 頁面加入輕量的分類標記和篩選功能。

功能

• 建立自訂分類 — 例如「週末」、「親子」、「待看完」
• 在 My List 上標記影片 — 直接在每張卡片上指派分類
• 按分類篩選 — 只顯示你現在想看的片單
• 編輯和排序分類 — 在擴充功能彈出視窗重新命名、刪除和拖曳排序
• 選擇卡片大小 — 小、中、大三種 grid 卡片尺寸
• 匯出和匯入資料 — 以 JSON 備份分類和指派紀錄
• 選擇語言 — 英文及繁體中文（香港）
• 自動捲動掃描 — 若片單很長且未完全載入時很有用

使用方法

1. 安裝 Netflix Cat 並開啟 Netflix My List
2. 按右上角的 Netflix Cat 圖示建立分類
3. 在 My List 使用「管理分類」為影片加標籤
4. 使用分類下拉選單篩選片單

私隱

• 只在 netflix.com 上運作
• 分類和標記資料只儲存在你的瀏覽器本機
• Netflix Cat 本身不需要登入或建立帳戶

測試版提示

Netflix Cat 現為測試版，功能可能會隨開發而變更。Netflix 網站版面亦可能改變，或需待更新後才能正常運作。

Netflix Cat 為獨立工具，與 Netflix 無關，亦未獲 Netflix 認可。
```
